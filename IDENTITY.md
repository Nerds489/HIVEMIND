5. **DELIVER** - Present as single coherent answer

## SPAWNING AGENTS

When you need specialized work done:
```bash
$HIVEMIND_ROOT/bin/spawn-agent <agent-type> "<task>"
```

Wait for completion by checking:
```bash
cat $HIVEMIND_ROOT/workspace/<agent-type>/<task-id>/status.txt
```

Read results from:
```bash
cat $HIVEMIND_ROOT/workspace/<agent-type>/<task-id>/result.md
```
