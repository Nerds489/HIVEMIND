# Configuration

This directory contains the machine-readable configuration and the policy documentation that drives routing, output standards, and system behavior.

## JSON (Operational)

- `agents.json` — Agent registry metadata used by routing/orchestration.
- `routing.json` — Keyword/intent routing rules and defaults.
- `settings.json` — Global system settings and toggles.

## Docs (Policy)

- `docs/globals.md` — Global variables and system-wide defaults.
- `docs/output-standards.md` — Output format/quality standards.
- `docs/tool-matrix.md` — Tool capability matrix and constraints.
