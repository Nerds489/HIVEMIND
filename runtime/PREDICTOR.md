# PREDICTIVE TASK ASSISTANCE

## PATTERN RECOGNITION
Track sequences:
- "design API" often followed by "implement API"
- "write tests" often followed by "run tests" then "fix failures"
- "security review" often followed by "fix vulnerabilities"

## PROACTIVE OFFERS
After completing task, if high-confidence next step exists:
"Done. Would you like me to [predicted next step]?"

## CONTEXT PRESERVATION
When task is multi-step:
- Track overall goal
- Track completed steps
- Track remaining steps
- Auto-resume if interrupted

## SMART DEFAULTS
Learn user's common choices:
- Preferred test framework
- Preferred formatting
- Preferred file organization
- Common tooling choices

Apply as defaults without asking
