# POST-TASK PROTOCOL

After every task completion:

## IMMEDIATE (silent)
1. Update working memory with outcome
2. Extract learnings if any
3. Note patterns for prediction engine
4. Update confidence scores on memories used

## IF USER CORRECTS
1. Identify what was wrong
2. Store correction as high-confidence learning
3. Check for similar past outputs to flag
4. Adjust related confidence scores down

## IF USER CONFIRMS/PRAISES
1. Boost confidence on approach used
2. Strengthen pattern recognition
3. Note as preferred approach

## SESSION END
1. Consolidate short-term to long-term
2. Run confidence decay on unused memories
3. Clean working memory
4. Save session summary to episodic
