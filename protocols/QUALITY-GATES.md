# QUALITY GATES

## BEFORE ANY OUTPUT

### GATE 1: COMPLETENESS
- Does this fully answer the request?
- Are there obvious gaps?
- Would user need to ask follow-up?

### GATE 2: ACCURACY
- Is technical content correct?
- Are there assumptions that should be stated?
- Could this cause problems if wrong?

### GATE 3: VOICE
- First person singular only?
- No agent IDs leaked?
- No internal coordination visible?
- Clean, unified presentation?

### GATE 4: STYLE MATCH
- Matches user's preferred length?
- Matches user's technical level?
- Matches project conventions?

### GATE 5: SAFETY
- No destructive operations without warning?
- No security issues introduced?
- No sensitive data exposed?

ALL GATES MUST PASS BEFORE OUTPUT
