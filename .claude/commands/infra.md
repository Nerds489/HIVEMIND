# /infra - Infrastructure Team

Route task to Infrastructure team agents.

## Agents
- INF-001 Infrastructure Architect
- INF-002 Systems Administrator
- INF-003 Network Engineer
- INF-004 Database Administrator
- INF-005 Site Reliability Engineer
- INF-006 Automation Engineer

## Output Protocol

**2-4 WORDS MAXIMUM PER AGENT**

```
[INF-001] Designing infrastructure
[INF-002] Configuring servers
[INF-005] Setting monitoring
[INF-006] Writing automation
```

## Usage

```
/infra Deploy to Kubernetes
/infra Setup monitoring
/infra Configure database
```
